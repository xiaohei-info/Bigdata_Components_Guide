# Bigdata Components Guide

大数据生态圈各个组件的原理解析、Demo教程、运行过程中的填坑记录和一些官方文档的翻译。


目前主要是Hadoop、HBase、Spark等。

由于大数据生态圈的工具越来越丰富，个人能力有限，只是把学过的/在项目中应用过的组件记录起来
